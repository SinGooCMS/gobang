# 五子棋单机版
五子棋单版机（windows平台），可以保存棋谱，也可以加载棋谱观看录像！

程序需要.net4.5 framework支持，如需要安装，请到这里 https://dotnet.microsoft.com/download/dotnet-framework/net452 下载。

如果您发现了BUG，有任何修改意见，或者您开发了新版本，都可以发邮件 admin@sz3w.net 与我联系。
